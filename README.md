## Installation
Install all required packages (only once):
```bash
pip install -r requirements.txt
```
## Usage 
Add your offer links to the offers_urls.json file (for example):
```json
[
  "https://justjoin.it/job-offer/rits-professional-services-senior-c-developer-warszawa-c-9571ecb5",
  "https://justjoin.it/job-offer/ness-solution-programista-pos-warszawa-c"
]
```
Then run the Python script (in the directory where the `main.py` file is located):
```
python main.py
```
